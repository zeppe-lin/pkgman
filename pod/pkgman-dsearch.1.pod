=head1 NAME

pkgman-dsearch - search the packages sources directories for package
name and description

=head1 SYNOPSIS

B<pkgman> dsearch [B<-v>|B<-vv>] [B<--path>] [B<--regex>] I<EXPR>

=head1 DESCRIPTION

Search the packages sources directories for I<EXPR> in packages'
name B<and> description.  It looks in all directories specified in
F</etc/pkgman.conf> as C<pkgsrcdir> directories.

=head1 OPTIONS

=over 8

=item B<-v>

Show package version in the listing.

=item B<-vv>

Show package version and description in the listing.

=item B<--path>

Show package source path in the listing.

=item B<--regex>

Interpret I<EXPR> as regular expression.

=item I<EXPR>

Search names and descriptions for I<EXPR>, where the expression may be
the string or regex in L<regex(3)> format.

=back

=head1 EXIT STATUS

=over 8

=item 0

Success.

=item 255

No matching packages found.

=back

=cut
