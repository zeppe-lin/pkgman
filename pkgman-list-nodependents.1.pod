=head1 NAME

pkgman-list-nodependents - print list of packages that no other depends on them

=head1 SYNOPSIS

B<pkgman> list-nodependents [B<-v>|B<-vv>] [B<--path>]

=head1 DESCRIPTION

Print list of packages without dependents. "Nodependent" are packages without
packages depending on them.

=head1 OPTIONS

=over 8

=item B<-v>

Show version in the listing.

=item B<-vv>

Show version and description in the listing.

=item B<--path>

Show package' source path in the listing.

=back

=cut
